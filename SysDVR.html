Стрим экрана без карты захвата

Системный модуль SysDVR позволяет транслировать изображение и звук с Nintendo Switch прямо на ваш компьютер по USB-кабелю или Wi-Fi для записи в OBS или стриминга в Discord!

Как использовать SysDVR

Ограничение: SysDVR транслирует картинку только в разрешении 720p @ 30 FPS. Видео записывается из видеобуфера самой консоли. Некоторые тяжелые игры могут отключать возможность захвата для экономии ресурсов.
  • Подготовка Свича. В сборках UltraNX и Kefir модуль SysDVR обычно уже установлен. Если нет — скачайте его с GitHub и закиньте папку atmosphere в корень вашей флешки. Перезагрузите консоль.
  • Подготовка ПК (Клиент). Скачайте на компьютер клиентскую программу SysDVR-Client (GUI-версию для удобства).
  • Запуск на консоли. Зайдите в Homebrew Menu (через Альбом) и откройте приложение SysDVR Settings. Выберите режим:
    • USB (Рекомендуется): Идеальное качество, без пинга. Подключите Switch к ПК кабелем Type-C.
    • TCP Bridge (Wi-Fi): Беспроводной стрим. Требует хорошего роутера (5GHz). Запомните IP-адрес, который покажет консоль.
  • Прием на ПК. Откройте SysDVR-Client GUI на компьютере.
    • Выберите Stream Source: USB или сеть (введите IP адрес консоли).
    • Нажмите Launch.
  • Результат. На вашем ПК откроется окно плеера (mpv/vlc), где будет в реальном времени отображаться геймплей со Switch вместе со звуком!
  • Стрим в Discord/OBS. Теперь вы можете просто захватить окно этого плеера в программе OBS Studio или нажать "Демонстрация экрана -> Окно" в Discord, чтобы показать игру друзьям!
